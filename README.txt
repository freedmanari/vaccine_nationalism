
GENERAL INFORMATION
This README.txt file was updated on March 18, 2025

Data
- sim_results/*, gives the output from simulations taking longer than ~10 minutes to run, these text files can be loaded in to vaccine_nationalism.ipynb to skip long computations if desired

Figures
- figures/*, final pdfs of all figures

Scripts
- vaccine_nationalism.ipynb, main script, can run all simulations and make all figures
- Cs.py, optional script to run very long computation to create Cs.txt, figures/f_Bs.pdf, and figures/f_Bs_no_cost.pdf; written for parallelization on HPC
